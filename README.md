## 📂 Project Structure

| File | Description |
|------|------------|
| 🧮 `kNN_iris/` | k-Nearest Neighbors applied to the Iris dataset |
| 🐸 `frog-leap.py` | Frog Leap puzzle solver (state-space search) |
| 🎒 `kp.py` | Knapsack problem implementation |
| ♛ `n-queens.py` | N-Queens solver using the **min-conflicts algorithm** |
| 🗳️ `naive-bayes-voting.py` | Naive Bayes classifier for voting records |
| 🧩 `npuzzle.py` | N-Puzzle solver using **IDA\*** |
| ❌⭕ `tictactoe.py` | Tic-Tac-Toe AI implementation |

---

## 🚀 Algorithms Included

### 🤖 Machine Learning
- k-Nearest Neighbors (kNN)
- Naive Bayes Classifier

### 🔍 Search & Optimization
- IDA* (Iterative Deepening A*)
- Min-Conflicts Algorithm
- State-Space Search

### 🧠 Classic Problems
- N-Queens Problem
- Knapsack Problem
- N-Puzzle
- Tic-Tac-Toe AI
- Frog Leap Puzzle
